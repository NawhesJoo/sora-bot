# 🐚 마법의 소라고동 디스코드 봇

스폰지밥의 마법의 소라고동이 당신의 질문에 답해드립니다!

![Discord](https://img.shields.io/badge/Discord.js-v14-5865F2?style=flat-square&logo=discord&logoColor=white)
![Node.js](https://img.shields.io/badge/Node.js-v16+-339933?style=flat-square&logo=node.js&logoColor=white)

## 📖 소개

마법의 소라고동 봇은 스폰지밥 애니메이션에 등장하는 "마법의 소라고동"을 모티브로 한 재미있는 디스코드 봇입니다. 
질문을 하면 소라고동이 랜덤하게 답변을 해줍니다!

## ✨ 주요 기능

- 🎲 **랜덤 답변**: 긍정, 부정, 애매한 답변 중 랜덤으로 응답
- 💬 **존댓말 필터**: '요?' 또는 '까?'로 끝나는 존댓말 질문만 받아들임
- ⚡ **슬래시 커맨드**: Discord의 슬래시 커맨드 지원

## 🚀 설치 방법

### 1. 사전 요구사항

- Node.js v16.9.0 이상
- Discord 봇 토큰
- Discord Developer Portal에서 봇 생성

### 2. 프로젝트 설정

```bash
# 저장소 클론
git clone <repository-url>
cd magic-conch-bot

# 패키지 설치
npm install
```

### 3. 환경 설정

`config.json` 파일을 생성하고 다음 내용을 입력하세요:

```json
{
  "token": "YOUR_BOT_TOKEN_HERE",
  "clientId": "YOUR_CLIENT_ID_HERE"
}
```

### 4. 봇 실행

```bash
node index.js
```

## 📋 사용 방법

### 명령어

| 명령어 | 설명 |
|--------|------|
| `/마법의소라고동님 질문:[질문]` | 마법의 소라고동님께 질문하기 |
| `/소라고동님 질문:[질문]` | 소라고동님께 질문하기 |
| `/도움말` | 사용법 안내 보기 |

### 사용 조건

⚠️ **중요한 규칙**

1. **존댓말 사용**: 질문은 '요?' 또는 '까?'로 끝나야 합니다
2. **물음표**: 물음표 개수는 상관없습니다 (?, ??, ??? 모두 가능)

### 사용 예시

✅ **올바른 사용법**
```
/소라고동님 질문:오늘 치킨 먹어도 될까요?
/마법의소라고동님 질문:숙제 해야 할까요???
/소라고동님 질문:내일 날씨 좋을까요?
```

❌ **잘못된 사용법**
```
/소라고동님 질문:치킨 먹어도 돼?        (반말)
/소라고동님 질문:어떻게 할까요          (물음표 없음)
```

## 🎭 답변 종류

소라고동은 다음과 같은 답변을 랜덤하게 제공합니다:

### 긍정 답변
- "그래"
- "어"
- "그러렴"
- "ㄱ"

### 부정 답변
- "아니"
- "안 돼"
- "절.대.안.돼"
- "ㄴ"

### 애매한 답변
- "흠..."
- "마음대로 해"
- "...아무것도 하지 마..."
- "ㅁㄹ?"

### 무례한 질문에 대한 답변
- "(무시)"
- "존댓말로 다시해."

## 🛠️ 커스터마이징

### 답변 추가/수정

`index.js` 파일의 `answers` 배열을 수정하여 답변을 추가하거나 변경할 수 있습니다:

```javascript
const answers = [
  // 긍정 답변
  "그래",
  "어",
  
  // 부정 답변
  "아니",
  "안 돼",
  
  // 애매한 답변
  "흠...",
  "마음대로 해",
  
  // 여기에 새로운 답변 추가!
];
```
