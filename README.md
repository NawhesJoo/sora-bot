# 🐚 마법의 소라고동 봇

스폰지밥의 마법의 소라고동이 Discord에 왔어요!

## 📖 소개

스폰지밥 에피소드에 등장하는 "마법의 소라고동"을 Discord 봇으로 구현했습니다.  
고민이 있을 때 소라고동에게 물어보세요!

## ✨ 기능

- 🎲 랜덤 답변 생성
- 🐚 원작 오마주 대사 포함
- ⚡ 빠른 응답
- 💬 간편한 명령어

## 🚀 설치 방법

### 1. 저장소 클론

```bash
git clone <repository-url>
cd magic-conch
```

### 2. 의존성 설치

```bash
npm install
```

### 3. 설정 파일 작성

`config.json` 파일을 수정하세요:

```json
{
  "token": "YOUR_DISCORD_BOT_TOKEN",
  "clientId": "YOUR_BOT_CLIENT_ID"
}
```

### 4. 봇 실행

```bash
npm start
```

## 📝 명령어

| 명령어 | 설명 | 예시 |
|--------|------|------|
| `/소라고동 질문:[질문]` | 소라고동에게 질문하기 | `/소라고동 질문:오늘 치킨 먹어도 돼?` |
| `/소라고동아 질문:[질문]` | 짧은 명령어 | `/소라고동아 질문:숙제 해야 돼?` |
| `/도움말` | 사용법 보기 | `/도움말` |

## 💡 사용 팁

1. **Yes/No 질문이 best!**
   - ✅ "오늘 피자 먹어도 돼?"
   - ✅ "숙제 미루면 안 돼?"
   - ✅ "게임해도 괜찮을까?"

2. **재미있게 사용하세요!**
   - 진지한 결정은 본인이 하세요 😄
   - 소라고동의 답은 랜덤이에요

3. **원하는 답이 안 나오면?**
   - 다시 물어보세요!
   - 소라고동도 기분이 있답니다 🐚

## 🎨 커스터마이징

### 답변 추가하기

`index.js` 파일의 `answers` 배열에 새로운 답변을 추가하세요:

```javascript
const answers = [
  "그래",
  "안 돼",
  "당신이 추가한 답변",  // 여기에 추가!
  // ...
];
```

### 답변 지연 시간 조정

```javascript
setTimeout(async () => {
  await interaction.editReply({
    content: `🐚 **마법의 소라고동**\n\n질문: *${question}*\n\n> ${answer}`,
  });
}, 1000); // 1000 = 1초, 원하는 시간(밀리초)으로 변경
```
